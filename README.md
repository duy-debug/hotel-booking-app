# Hotel Booking & Management System

A comprehensive web application for hotel operations and customer reservations, built with ASP.NET MVC.

## Key Features

### Admin Management
- **User Management:** Control accounts for both administrators and customers.
- **Room & Category Management:** Create and update rooms, room types, and maintain availability.
- **Service Management:** Manage additional hotel services and their pricing.
- **Booking Operations:** Handle offline bookings, check-ins, check-outs, and status updates.
- **Payment & Invoicing:** Process payments (Cash/Transfer) and generate financial records.
- **Reports & Analytics:** Detailed statistics on revenue, occupancy rates, and room status.

### Customer Experience
- **Online Booking:** Search, filter, and reserve rooms effortlessly.
- **Personal Dashboard:** Manage profile, booking history, and active services.
- **Service Requests:** Browse and add services to existing bookings.
- **Invoicing:** View payment history and download/print PDF invoices.

## Technology Stack
- **Backend:** C# (.NET Framework), ASP.NET MVC
- **Data Access:** Entity Framework
- **Database:** Microsoft SQL Server
- **Frontend:** HTML5, CSS3 (Modern UI), JavaScript/jQuery
